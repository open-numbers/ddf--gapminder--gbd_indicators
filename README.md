# ddf--gapminder--gbd_indicators

Indicators calculated using IHME GBD dataset

These indicators are part of [Systema Globalis](https://github.com/open-numbers/ddf--gapminder--systema_globalis).

# How to Update

To update this dataset, first update the source dataset [ddf--ihme--death_cause_compact](https://github.com/open-numbers/ddf--ihme--death_cause_compact) and run the etl recipe in this dataset.
